<p><strong>Terrain Toggle</strong></p>
<p>A lightweight version of the blindfold plugin made by stevenwaterman to remove the floors.</p>
<p>Includes:</p>
<ul>
<li>Global toggling</li>
<li>Hotkey toggling</li>
<li>Region automation
<ul>
<li>Hide/Show selected regions</li>
<li>Remembers instances and raid rooms</li>
<li>Shift+Right click menu option to add current region to hide/show list</li>
</ul>
</li>
</ul>
<p>The skybox colour is displayed in place of removed terrain, you can update the colour in the skybox plugin to change it.</p>
<p>Must be used in combination with RuneLiteâ€™s GPU plugin.</p>
<ul>
<li>Does not currently support 117 HD.</li>
</ul>
